**GORM struct + tag pattern** (`stores.go:66-77`):
```go

    ID        string         `gorm:"primaryKey;size:36" json:"id"`
    Name      string         `gorm:"uniqueIndex;not null;size:255" json:"name"`
    Kind      BlockStoreKind `gorm:"not null;size:10;index" json:"kind"`
```

Apply directly to `BackupRepo`. Notes for Phase 1:
- `ID string size:36` for UUID-style IDs (BackupRepo) — matches existing convention. For ULIDs (`BackupRecord`, `BackupJob`), keep `size:36` (ULID canonical form is 26 chars, fits) but the value is generated by `oklog/ulid/v2` rather than `uuid.New()`.
- Composite uniqueness `(metadata_store_id, name)` for `BackupRepo` — use GORM `uniqueIndex:idx_backup_repo_store_name` on both columns instead of single-column `uniqueIndex`.
- FK relationships use the `Share` precedent (`share.go:40-42`):
  ```go
  MetadataStoreID string                  `gorm:"not null;size:36" json:"metadata_store_id"`
  MetadataStore   MetadataStoreConfig     `gorm:"foreignKey:MetadataStoreID" json:"metadata_store,omitempty"`
  ```
- Nullable FK (`backup_record_id` on `BackupJob` when `kind=backup`) uses pointer pattern from `share.go:25`:
  ```go
  RemoteBlockStoreID *string `gorm:"size:36" json:"remote_block_store_id"`
  ```

**Opaque destination config (Get/SetConfig)** (`stores.go:84-108`) — apply to `BackupRepo.Config` (destination-specific JSON blob: e.g. local path, S3 bucket/region/prefix). Copy verbatim; rename receiver.

```go
// Package metadata's Backupable interface (capability check via type assertion).
//
// Stores that support backup/restore implement Backupable. Stores that don't
// must not implement it; callers do `if b, ok := store.(Backupable); ok {...}`
// or surface ErrBackupUnsupported.
type Backupable interface {
    Backup(ctx context.Context, w io.Writer) (PayloadIDSet, error)
    Restore(ctx context.Context, r io.Reader) error
}

// PayloadIDSet is the set of block PayloadIDs referenced by a snapshot.
// Used by the GC hold path (SAFETY-01).
type PayloadIDSet map[string]struct{}

func (s PayloadIDSet) Add(id string)         { s[id] = struct{}{} }
func (s PayloadIDSet) Contains(id string) bool { _, ok := s[id]; return ok }
func (s PayloadIDSet) Len() int               { return len(s) }
```

**Struct shape** (locked in CONTEXT.md):
```go
package manifest

import (
    "fmt"
    "io"
    "time"

    "gopkg.in/yaml.v3"
)

// CurrentVersion is the manifest schema version produced by this build.
const CurrentVersion = 1

// Encryption describes payload-at-rest encryption parameters.
// Never contains the key itself — only a reference resolvable at restore time.
type Encryption struct {
    Enabled   bool   `yaml:"enabled"`
    Algorithm string `yaml:"algorithm,omitempty"` // e.g. "aes-256-gcm"
    KeyRef    string `yaml:"key_ref,omitempty"`   // env var name or file path
}

// Manifest is the self-describing, versioned descriptor written alongside every backup payload.
type Manifest struct {
    ManifestVersion int               `yaml:"manifest_version"`
    BackupID        string            `yaml:"backup_id"`         // ULID
    CreatedAt       time.Time         `yaml:"created_at"`        // RFC3339 via yaml.v3 default
    StoreID         string            `yaml:"store_id"`          // FK snapshot
    StoreKind       string            `yaml:"store_kind"`        // memory|badger|postgres
    SHA256          string            `yaml:"sha256"`
    SizeBytes       int64             `yaml:"size_bytes"`
    Encryption      Encryption        `yaml:"encryption"`
    PayloadIDSet    []string          `yaml:"payload_id_set"`    // sorted list (deterministic output)
    EngineMetadata  map[string]string `yaml:"engine_metadata,omitempty"`
}

// Marshal serializes the manifest to YAML.
func (m *Manifest) Marshal() ([]byte, error) {
    return yaml.Marshal(m)
}

// WriteTo streams the YAML encoding to w.
func (m *Manifest) WriteTo(w io.Writer) (int64, error) {
    data, err := m.Marshal()
    if err != nil {
        return 0, err
    }
    n, err := w.Write(data)
    return int64(n), err
}

// Parse decodes a YAML manifest and validates its version.
func Parse(data []byte) (*Manifest, error) {
    var m Manifest
    if err := yaml.Unmarshal(data, &m); err != nil {
        return nil, fmt.Errorf("decode manifest: %w", err)
    }
    if err := m.Validate(); err != nil {
        return nil, err
    }
    return &m, nil
}

// ReadFrom decodes a YAML manifest from r.
func ReadFrom(r io.Reader) (*Manifest, error) {
    data, err := io.ReadAll(r)
    if err != nil {
        return nil, fmt.Errorf("read manifest: %w", err)
    }
    return Parse(data)
}

// Validate enforces structural invariants and version compatibility.
// Phase 1 only accepts manifest_version == CurrentVersion. Future versions
// will branch here for forward-compat handling (SAFETY-03).
func (m *Manifest) Validate() error {
    if m.ManifestVersion == 0 {
        return fmt.Errorf("manifest_version is required")
    }
    if m.ManifestVersion != CurrentVersion {
        return fmt.Errorf("unsupported manifest_version %d (this build supports %d)", m.ManifestVersion, CurrentVersion)
    }
    if m.BackupID == "" {
        return fmt.Errorf("backup_id is required")
    }
    if m.StoreID == "" {
        return fmt.Errorf("store_id is required")
    }
    if m.StoreKind == "" {
        return fmt.Errorf("store_kind is required")
    }
    if m.SHA256 == "" {
        return fmt.Errorf("sha256 is required")
    }
    return nil
}
```

### Generic GORM helpers (already in place)

**Source:** `pkg/controlplane/store/helpers.go`
**Apply to:** All `BackupStore` operations.

Available:
- `getByField[T](db, ctx, field, value, notFoundErr, preloads...)` — single-record lookup with domain error mapping (`helpers.go:21-31`).
- `listAll[T](db, ctx, preloads...)` — slice of all records (`helpers.go:39-49`).

- `convertNotFoundError(err, notFoundErr)` (`gorm.go:330-335`) — wraps `gorm.ErrRecordNotFound` → domain error.
- `isUniqueConstraintError(err)` (`gorm.go:319-327`) — SQLite + Postgres unique-violation detection.
