    **1. Create `pkg/metadata/lock/leases.go`** — Lease CRUD implementation:

    All methods are on `*Manager` receiver.

    **RequestLease:**
    - Acquire lm.mu.Lock()
    - Validate requestedState: if isDirectory, check `IsValidDirectoryLeaseState()`; if file, check `IsValidFileLeaseState()`. Return (LeaseStateNone, 0, nil) for invalid.
    - If isDirectory, check recently-broken cache. If recently broken, return (LeaseStateNone, 0, nil).
    - Call CheckNLMLocksForLeaseConflict if lockStore is set. If conflicts, return (LeaseStateNone, 0, nil).
    - Search existing leases: iterate `lm.unifiedLocks[string(fileHandle)]` for locks with matching leaseKey.
    - **Same-key found (upgrade/maintain):** Validate upgrade transition is allowed. Allowed: state increase (R->RW, R->RH, RH->RWH, etc.). Disallowed: downgrade (RWH->R). If same state, return current. If valid upgrade, update lease state, increment epoch, persist, return new state and epoch.
    - **Different-key conflict:** Use `OpLocksConflict()` to check. If conflict, calculate breakToState, call `lm.breakOpLocks()` to initiate break, return (LeaseStateNone, 0, nil) -- caller retries after break.
    - **No conflict:** Create new UnifiedLock with Lease field. Set Epoch=1, IsDirectory, ParentLeaseKey. Persist via lockStore.PutLock. Add to unifiedLocks map. Return (requestedState, 1, nil).

    **AcknowledgeLeaseBreak:**
    - Find lease by leaseKey across all handleKeys.
    - Validate: must be Breaking=true. If not, return error.
    - Validate epoch: if epoch != 0, check Lease.Epoch+1 == provided epoch (V2 staleness check). If stale, return error.
    - Validate acknowledgedState <= breakToState (client cannot claim more than offered).
    - Set Lease.LeaseState = acknowledgedState, Lease.Breaking = false, Lease.BreakToState = 0.
    - Increment Lease.Epoch.
    - If acknowledgedState == LeaseStateNone, remove the lock entirely (lease fully released).
    - Persist updated state.

    **ReleaseLease:**
    - Find all locks with matching leaseKey across all handleKeys.
    - Remove them from in-memory maps. Delete from lockStore.
    - Clean up empty handleKey entries.

    **GetLeaseState:**
    - Search all handleKeys for a lock with matching leaseKey.
    - Return (state, epoch, true) or (0, 0, false).

    **Helper: findLeaseByKey** — scans unifiedLocks for a specific leaseKey. Returns (handleKey, *UnifiedLock) or empty.

    **Helper: advanceEpoch** — centralizes epoch increment: `lock.Lease.Epoch++`. Called from grant, break initiate, break ack, upgrade. EVERY state change goes through this.

    **Upgrade transition validation:**

    var validUpgrades = map[uint32][]uint32{
