**pkg/metadata/store.go — Add to FileBlockStore interface:**
```go
// ListFileBlocks returns all blocks belonging to a file, ordered by block index.
// Block IDs follow the format "{payloadID}/{blockIdx}", so this method returns
// all blocks whose ID starts with "{payloadID}/".
// Returns empty slice (not nil) if no blocks found.
ListFileBlocks(ctx context.Context, payloadID string) ([]*FileBlock, error)
```

**pkg/metadata/store/memory/objects.go:**
1. Add `ListFileBlocks` method on MemoryMetadataStore:
   - Iterate over `s.fileBlockData.blocks`
   - Filter blocks whose ID starts with `payloadID + "/"`
   - Parse blockIdx from the ID suffix for sorting
   - Sort results by block index (ascending)
   - Return copies (not pointers into the map)
2. Add locked helper `listFileBlocksLocked`
3. Add transaction wrapper on memoryTransaction:
   ```go
   func (tx *memoryTransaction) ListFileBlocks(ctx context.Context, payloadID string) ([]*metadata.FileBlock, error) {
       return tx.store.listFileBlocksLocked(ctx, payloadID)
   }
   ```

**pkg/metadata/store/badger/objects.go:**
1. Add new constant: `fileBlockFilePrefix = "fb-file:"`
2. In `PutFileBlock`: maintain a `fb-file:{payloadID}:{blockIdx}` secondary index entry.
   - Parse payloadID and blockIdx from `block.ID` (format: "{payloadID}/{blockIdx}")
   - The index key format: `fb-file:{payloadID}:{blockIdx}` with value = block.ID
   - Add the index entry on every PutFileBlock (all states, not just Local)
3. In `DeleteFileBlock`: remove the `fb-file:{payloadID}:{blockIdx}` entry
4. Add `ListFileBlocks` method:
   - Prefix scan on `fb-file:{payloadID}:` to get block IDs
   - Fetch each block from `fb:{id}`
   - Results are naturally ordered by block index since keys are lexicographically sorted
   - Append to result slice with copies
5. Add transaction wrapper:
   ```go
   func (tx *badgerTransaction) ListFileBlocks(ctx context.Context, payloadID string) ([]*metadata.FileBlock, error) {
       return tx.store.ListFileBlocks(ctx, payloadID)
   }
   ```

**pkg/metadata/store/postgres/objects.go:**
1. Add `ListFileBlocks` method:
   ```sql
   SELECT id, hash, data_size, cache_path, block_store_key, ref_count, last_access, created_at, state
   FROM file_blocks
   WHERE id LIKE $1
   ORDER BY id ASC
   ```
   Where `$1` is `payloadID + "/%"`.
   Note: Block IDs are "{payloadID}/{blockIdx}" and blockIdx is numeric, so ordering by id ASC
   gives correct ordering since all payloadID prefixes are the same within a query.
   However, for correct numeric ordering of block indices, use a more precise approach:
   Parse and sort in Go after fetching, OR use `ORDER BY LENGTH(id), id` for natural sort.
   Simplest correct approach: fetch all, sort in Go by parsed blockIdx.
2. Add transaction wrapper:
   ```go
   func (tx *postgresTransaction) ListFileBlocks(ctx context.Context, payloadID string) ([]*metadata.FileBlock, error) {
       return tx.store.ListFileBlocks(ctx, payloadID)
   }
   ```
