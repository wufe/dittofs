**File 1: test/e2e/smb3_gosmb2_test.go**

go-smb2 E2E tests covering the full SMB3 feature matrix:

1. **TestSMB3_GoSMB2_BasicFileOps**: Use SetupSMB3TestEnv, connect via ConnectSMB3, mount share, create file with WriteFile, read back with ReadFile, verify content matches, stat file verify size, rename file, verify renamed file readable, delete file, verify file gone.

2. **TestSMB3_GoSMB2_DirectoryOps**: Create directories with Mkdir, create files inside, ReadDir to list contents, verify entry names and types, Remove files, Rmdir directories, verify empty.

3. **TestSMB3_GoSMB2_LargeFile**: Generate 1MB random data, WriteFile, ReadFile, verify exact byte match. Tests the data path under SMB3.

4. **TestSMB3_GoSMB2_SessionSetup**: Connect with valid credentials, verify session established (non-nil session). Connect with invalid credentials, verify error. Connect with wrong password, verify error. This tests NTLM authentication.

5. **TestSMB3_GoSMB2_Encryption**: Set up DittoFS with encryption enabled on the share (via API: set share EncryptData=true or configure adapter encryption). Connect via go-smb2, perform file write/read, verify operations succeed. The go-smb2 library handles encryption transparently.

6. **TestSMB3_GoSMB2_Signing**: Set up DittoFS with signing required. Connect via go-smb2, perform file ops, verify success. go-smb2 handles signing transparently.

7. **TestSMB3_GoSMB2_MultipleFiles**: Create 50 files with unique names and content, ReadDir to list all, verify all 50 present. Read each file back and verify content. Tests directory enumeration under load.

Use table-driven subtests where appropriate. Each test should be independent and create its own test environment.

**File 2: test/e2e/smb3_smbclient_test.go**

smbclient exec tests:

1. **TestSMB3_SmbClient_Connect**: Use SetupSMB3TestEnv, run smbclient with `ls` command, verify exit code 0 and output contains share listing.

2. **TestSMB3_SmbClient_FileOps**: Create a temp file locally, use smbclient `put` to upload, `get` to download to different path, verify content matches. Use `del` to delete.

3. **TestSMB3_SmbClient_DialectNegotiation**: Run smbclient with `--debuglevel=1 --max-protocol=SMB3`, parse output for "Protocol negotiated:" line, verify SMB3 protocol (3.0, 3.0.2, or 3.1.1). Use minimal output parsing per research recommendation.

4. **TestSMB3_SmbClient_DirectoryOps**: Use smbclient `mkdir`, `cd`, `ls`, `rmdir` commands. Verify operations succeed.

For smbclient tests, use the RunSMBClient helper from Task 1. Check for smbclient binary availability and t.Skip if not installed. Parse output minimally: check exit code, look for NT_STATUS_ error strings, check for expected filenames.
