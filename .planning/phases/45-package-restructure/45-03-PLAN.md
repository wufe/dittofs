```go
type Config struct {
    Local         local.LocalStore
    Remote        remote.RemoteStore   // nil for local-only mode
    Syncer        *blocksync.Syncer    // import alias: blocksync "github.com/marmos91/dittofs/pkg/blockstore/sync"
    GC            *gc.Collector        // nil if GC not needed
}

type BlockStore struct {
    local   local.LocalStore
    remote  remote.RemoteStore
    syncer  *blocksync.Syncer
    gc      *gc.Collector
}

func New(cfg Config) (*BlockStore, error) { ... } // validate local is non-nil
func (bs *BlockStore) Start(ctx context.Context) error { ... } // recovery then syncer.Start
func (bs *BlockStore) Close() error { ... } // syncer.Close, local.Close, remote.Close if non-nil
```

Implement all methods from the Store interface:
- ReadAt: delegate to bsio.ReadAt (from io/ package)
- ReadAtWithCOWSource: delegate to bsio.ReadAtWithCOWSource
- WriteAt: delegate to bsio.WriteAt
- Truncate: local.Truncate + syncer.Truncate
- Delete: local.EvictMemory + syncer.Delete
- GetSize: local.GetFileSize first, fall back to syncer.GetFileSize
- Exists: local.GetFileSize first, fall back to syncer.Exists
- Flush: syncer.Flush
- DrainAllUploads: syncer.DrainAllUploads
- Stats: local.Stats + wrap in blockstore.Stats
- HealthCheck: syncer.HealthCheck
- Close: syncer.Close, local.Close, remote.Close (if non-nil)

Add interface satisfaction: `var _ Store = (*BlockStore)(nil)`

Add accessor methods for use by runtime/config code:
- `Local() local.LocalStore`
- `Remote() remote.RemoteStore`
- `Syncer() *blocksync.Syncer`

Use import alias: `blocksync "github.com/marmos91/dittofs/pkg/blockstore/sync"` to avoid collision with Go's sync.
