    **Part B: Create durable_context.go** following `lease_context.go` pattern:

    Constants:
    ```
    DurableHandleV1RequestTag  = "DHnQ"
    DurableHandleV1ReconnectTag = "DHnC"
    DurableHandleV2RequestTag  = "DH2Q"
    DurableHandleV2ReconnectTag = "DH2C"
    AppInstanceIdTag           = "4543" // SMB2_CREATE_APP_INSTANCE_ID
    ```

    Wire format functions (use `smbenc.NewReader`/`smbenc.NewWriter`):
    - `DecodeDHnQRequest(data []byte) error` -- 16 bytes reserved, validate length
    - `DecodeDHnCReconnect(data []byte) ([16]byte, error)` -- 16 bytes FileId
    - `DecodeDH2QRequest(data []byte) (timeout uint32, flags uint32, createGuid [16]byte, err error)` -- 32 bytes
    - `DecodeDH2CReconnect(data []byte) (fileID [16]byte, createGuid [16]byte, flags uint32, err error)` -- 36 bytes
    - `DecodeAppInstanceId(data []byte) ([16]byte, error)` -- 20 bytes
    - `EncodeDHnQResponse() CreateContext` -- DHnC tag, 8 bytes zeros
    - `EncodeDH2QResponse(timeoutMs uint32, flags uint32) CreateContext` -- DH2Q tag, 8 bytes
    - `EncodeDHnCResponse() CreateContext` -- for V1 reconnect response

    `ProcessDurableHandleContext(...)` function:
    - Params: durableStore, contexts []CreateContext, openFile *OpenFile, configuredTimeoutMs uint32
    - Check for DH2Q first (V2 precedence). If found:
      - Decode DH2Q, reject persistent flag, calculate granted timeout
      - Set openFile.IsDurable = true, openFile.CreateGuid = createGuid, openFile.DurableTimeoutMs = grantedTimeout
      - Return DH2Q response context
    - If only DHnQ: decode, check oplock level is Batch (0x09). If not, return nil (no grant).
      - Set openFile.IsDurable = true, openFile.DurableTimeoutMs = configuredTimeoutMs
      - Return DHnC response context
    - If neither: return nil

    `ProcessDurableReconnectContext(...)` function:
    - Params: durableStore, metaSvc, contexts []CreateContext, sessionID, username, sessionKeyHash, shareName, filename, clientGuid [16]byte
    - Determine V1 (DHnC) or V2 (DH2C), parse reconnect context
    - Look up handle: V1 by FileID, V2 by CreateGuid
    - Run validation checks (14+ from RESEARCH.md), each with debug logging:
      1. Handle exists in store
      2. No conflicting context tags (V1 with DH2Q/DH2C present, etc.)
      3. Share name matches
      4. File path matches filename
      5. Security context: username match
      6. Security context: session key hash match (WARNING-level log if mismatch after restart for V2)
      7. DesiredAccess matches
      8. ShareAccess matches
      9. If handle has lease: lease key match, lease context present in request
      10. If handle has oplock but no lease: oplock was Batch (V1) or any non-None (V2)
      11. Handle not already reconnected (Open.Session is null = disconnected)
      12. File still exists (call metaSvc.GetFile with MetadataHandle)
      13. File path unchanged
    - Each check: log at debug level with check name, pass/fail, expected vs actual
    - On failure: return specific NTSTATUS code per check
    - On success: build OpenFile from persisted handle, delete persisted handle, return OpenFile

    `ProcessAppInstanceId(...)` function:
    - Params: durableStore, handler *Handler, contexts []CreateContext, ctx context.Context
    - Find AppInstanceIdTag context, decode
    - Look up existing handles by AppInstanceId
    - For each found: force-close (reuse cleanup logic: release locks, flush caches, delete-on-close, delete from store)
    - Return the parsed AppInstanceId (or zero if not present)

    **Part C: Integrate into CREATE handler** in `create.go`:

      2. Check for DHnC/DH2C reconnect contexts. If present: call ProcessDurableReconnectContext. On success, register the restored OpenFile and return CREATE response with reconstituted attributes. On failure, return error response with NTSTATUS.
      3. If not reconnect: call ProcessDurableHandleContext to potentially grant durability. If response context returned, add to response contexts.
      4. If AppInstanceId was parsed and durability was granted, store appInstanceId in openFile
