func (s *MetadataService) SetQuotaForShare(shareName string, quotaBytes int64) {

**Step 3: Quota check in PrepareWrite**

In `pkg/metadata/io.go`, add quota check after the context check and before the file lookup in `PrepareWrite()` (around line 87, after `if err := ctx.Context.Err(); err != nil { return nil, err }`):

```go
// Quota enforcement: check if this write would exceed the share's quota.
// Must happen early for fast rejection. Truncate (shrink) is always allowed.
shareName := shareNameForHandle(handle)
quotaBytes := s.GetQuotaForShare(shareName)
if quotaBytes > 0 && newSize > 0 {
    store, storeErr := s.storeForHandle(handle)
    if storeErr == nil {
        currentUsed := store.GetUsedBytes()
        // Get current file size to compute delta
        // We need the file anyway for PrepareWrite, so fetch it here
        // and reuse below (avoiding double fetch)
        file, fileErr := store.GetFile(ctx.Context, handle)
        if fileErr == nil && file.Type == FileTypeRegular {
            currentFileSize := int64(file.Size)
            // Only enforce on size increase (truncate/shrink always allowed)
            if int64(newSize) > currentFileSize {
                delta := int64(newSize) - currentFileSize
                if currentUsed+delta > quotaBytes {
                    return nil, &StoreError{
                        Code:    ErrNoSpace,
                        Message: "share quota exceeded",
                    }
                }
            }
        }
    }
}
```

IMPORTANT: The quota check should integrate cleanly with the existing file fetch logic. Since PrepareWrite already fetches the file (for permission checks and cached file), refactor to reuse the fetched file for quota checking. Do NOT do a double GetFile call. The quota check should use the same file data that the rest of PrepareWrite uses.

**Step 4: Quota-aware GetFilesystemStatistics**

In `pkg/metadata/service.go`, modify `GetFilesystemStatistics()` to overlay quota:

```go
func (s *MetadataService) GetFilesystemStatistics(ctx context.Context, handle FileHandle) (*FilesystemStatistics, error) {
    store, err := s.storeForHandle(handle)
    if err != nil {
        return nil, err
    }
    stats, err := store.GetFilesystemStatistics(ctx, handle)
    if err != nil {
        return nil, err
    }

    // Apply quota overlay if configured
    shareName := shareNameForHandle(handle)
    quotaBytes := s.GetQuotaForShare(shareName)
    if quotaBytes > 0 {
        stats.TotalBytes = uint64(quotaBytes)
        if stats.UsedBytes > uint64(quotaBytes) {
            stats.AvailableBytes = 0
        } else {
            stats.AvailableBytes = uint64(quotaBytes) - stats.UsedBytes
        }
    }

    return stats, nil
}
```
