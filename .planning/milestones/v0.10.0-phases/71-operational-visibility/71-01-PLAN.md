**Registry struct** — follows MountTracker pattern:
```go
type Registry struct {
    mu      sync.RWMutex
    clients map[string]*ClientRecord  // keyed by ClientID
    ttl     time.Duration
    stopCh  chan struct{}
}
```

**Constructor**: `func NewRegistry(ttl time.Duration) *Registry` — default TTL 5 min if zero.

**Methods** (all thread-safe via RWMutex):
- `Register(record *ClientRecord)` — stores record, sets ConnectedAt and LastActivity to now if zero
- `Deregister(clientID string) *ClientRecord` — removes and returns record (nil if not found)
- `Get(clientID string) *ClientRecord` — returns copy or nil
- `UpdateActivity(clientID string)` — updates LastActivity to now
- `AddShare(clientID, share string)` — appends share if not already present
- `RemoveShare(clientID, share string)` — removes share from record
- `List() []*ClientRecord` — copy-on-read, all records
- `ListByProtocol(protocol string) []*ClientRecord` — filtered
- `ListByShare(share string) []*ClientRecord` — filtered by share membership
- `Count() int`
- `collect(filter func(*ClientRecord) bool) []*ClientRecord` — internal, copies records

**TTL Sweeper** (D-05, D-06, D-07):
- `StartSweeper(ctx context.Context)` — starts background goroutine with ticker at ttl/2 interval
- `sweep()` — under write lock, removes records where `time.Since(r.LastActivity) > ttl`
- `Stop()` — closes stopCh to terminate sweeper goroutine

Copy-on-read: `collect` must deep-copy Shares slice (append to nil slice).
