## Accomplishments

### Task 1: CREATE Edge Cases (48491a96)
- Replaced `strings.TrimPrefix(filename, "/")` with `strings.TrimLeft(filename, "/")` to handle multiple leading slashes (e.g., `\\foo` becomes `//foo` after backslash conversion)
- Extracted `normalizeCreatePath()` as a testable pure function
- Added strict create context blob validation in `decodeCreateContexts()`:
  - NameOffset + NameLength must be within buffer bounds
  - DataOffset + DataLength must be within buffer bounds
  - Next must be 8-byte aligned and within buffer (or 0 for last)
- Changed `decodeCreateContexts` signature to return `([]CreateContext, error)` -- malformed blobs now return error which propagates as STATUS_INVALID_PARAMETER
- Added `TestCreate_LeadingSlashNormalization` (9 cases) and `TestCreate_CreateContextBlobValidation` (7 cases)
- Verified mkdir visibility is already working (synchronous metadata commits)

### Task 2: ADS/Streams Conformance (3f7a3dbe)
- Added ADS entry filtering in `filterDirEntries()` -- entries with `:` in their name are now excluded from QUERY_DIRECTORY results
- Verified existing ADS/streams implementation handles all smbtorture test scenarios:
  - FileStreamInformation: default stream + named streams with correct allocation sizes
  - Create disposition: OPEN, CREATE, OPEN_IF, OVERWRITE, OVERWRITE_IF, SUPERSEDE all work on streams
  - DELETE_ON_CLOSE: removes only the stream, not the base file
  - I/O: per-stream PayloadID for separate block store content
  - Name enumeration: `::$DATA` as first entry, `:streamname:$DATA` for named streams
  - Rename: stream-to-stream rename via SET_INFO FileRenameInformation
  - Share modes: cross-stream enforcement (base file + all streams)
  - Zero-byte: creatable and queryable with StreamSize=0
  - `::$DATA` suffix stripping: opening `file::$DATA` resolves to base file
- Removed 18 tests from KNOWN_FAILURES.md:
  - 14 streams tests (attributes1/2, basefile-rename-with-open-stream, create-disposition, delete, dir, io, names/2/3, rename2, sharemodes, zero-byte, create_no_streams.no_stream)
  - 3 create tests (leading-slash, mkdir-visible, blob)
  - 1 durable-v2 test (create-blob)
