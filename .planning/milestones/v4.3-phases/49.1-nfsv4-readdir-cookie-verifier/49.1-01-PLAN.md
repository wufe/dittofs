<objective>
Implement proper mtime-based cookie verifier for NFSv4 READDIR, replacing hardcoded 8 zero bytes.

Purpose: Prevents macOS Finder error -8062 during folder copy and improves RFC 7530 conformance by providing meaningful verifiers that clients can use to detect directory modifications between paginated READDIR calls. Mirrors the established NFSv3 implementation pattern.

Output: Updated readdir.go with verifier generation + advisory validation, updated tests asserting non-zero verifiers and mismatch behavior.
</objective>
